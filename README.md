# nest-wrapper-kiwoom

키움증권 REST API 연동을 위한 NestJS 모듈입니다.

## 특징

- 키움증권 공식 REST API 지원
- NestJS 모듈 시스템과 Service 주입 방식
- 타입 지원 및 에러 핸들링

## 설치

npm install nest-wrapper-kiwoom

## 시작하기

### API 키 발급

- 키움증권 계좌 필요
- 키움 오픈API 센터에서 REST API 사용 신청과 API Key, Secret Key 발급
